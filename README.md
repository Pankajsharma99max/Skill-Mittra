# Skill Mittra - Futuristic Robotics & IoT Learning Platform

Stack: Next.js (App Router) + TailwindCSS + Framer Motion

## Quick Start
1. Copy `.env.example` to `.env` and fill values.
2. Install deps: `npm install`
3. Dev server: `npm run dev` then open http://localhost:3000

## Theme
- Background: #0A0F1F
- Neon Blue: #4C8BFF
- Neon Purple: #A020F0
- Text: #E4E6EB

Fonts via next/font: Orbitron (headings), Inter (body), Montserrat (alt).

## Structure
- app/ routes match the sitemap (Shop, Learning Hub, Project Library, Workshops, About, Contact, Login, Signup, Admin)
- components/ shared UI (Navbar, Footer, GlassCard, ParticleBg)
